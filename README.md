# Financial Planning & Analysis (FP&A) Data Analytics Portfolio

## Executive Summary
Welcome to my FP&A Analytics Portfolio. I am Jericho James Largo Cainhog, a Certified Public Accountant (CPA) and Senior Financial Analyst. 

This repository bridges the gap between traditional corporate finance and advanced data analytics. Designed for executive review, the projects contained here demonstrate how complex financial datasets can be transformed into strategic insights using SQL. The focus is on automating variance analysis, optimizing cash flow visibility, and building robust predictive models.

## Core Competencies Demonstrated
* **Corporate Reporting:** Automating SEC public reporting and internal OPEX variance controls.
* **Predictive Modeling:** Utilizing time-series analysis for retail demand planning and financial risk assessment.
* **Subscription Economics:** Tracking and analyzing SaaS metrics including MRR, ARR, and churn.
* **Liquidity Analysis:** Processing large-scale banking transaction data for cash flow optimization.
